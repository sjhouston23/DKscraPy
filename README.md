# DKscraPy
Scrape Draftkings Sportsbook using Python's Beautiful Soup library and DraftKings' API.

## Available Scrapers:
* MLB World Series Odds
* MLB Game Moneyline Odds
* NFL Super Bowl Odds
* NFL Game Spreads, Totals and Moneylines
* NFL Player Props
